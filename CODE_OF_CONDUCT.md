# 贡献者公约行为准则

## 我们的承诺

作为成员、贡献者和领导者，我们承诺让每个人在我们的社区中免受骚扰，无论年龄、体型、可见或不可见的残疾、种族、性别特征、性别认同和表达、经验水平、教育程度、社会经济地位、国籍、个人外貌、种族、宗教或性取向如何。

我们承诺以有助于构建开放、欢迎、多元、包容和健康社区的方式行事和互动。

## 我们的标准

有助于为我们社区营造积极环境的行为示例包括：

*   对他人表现出同理心和善意
*   尊重不同的意见、观点和经验
*   给予并优雅地接受建设性反馈
*   承担责任并向受我们错误影响的人道歉，同时从经验中学习
*   关注什么对我们个人最好，但对整个社区最好

不可接受的行为示例包括：

*   使用性暗示的语言或图像，以及任何形式的性关注或挑逗
*   恶意攻击、侮辱或贬损性评论，以及人身或政治攻击
*   公开或私下的骚扰
*   未经他人明确许可发布他人的私人信息，如物理地址或电子邮件地址
*   其他在专业环境中可被合理认为不适当的行为

## 执行责任

社区领导者负责澄清和执行我们的标准，并将针对任何他们认为不适当、威胁性、攻击性或有害的行为采取适当和公平的纠正措施。

社区领导者有权利和责任删除、编辑或拒绝不符合本行为准则的评论、提交、代码、wiki 编辑、问题和其他贡献，并在适当时传达审核决定的原因。

## 适用范围

本行为准则适用于所有社区空间，也适用于个人在公共空间正式代表社区时。代表我们社区的示例包括使用官方电子邮件地址、通过官方社交媒体账户发布，或在在线或线下活动中担任指定代表。

## 执行

如有滥用、骚扰或其他不可接受的行为，可向 [fuqihan1@outlook.com](mailto:fuqihan1@outlook.com) 举报给负责执行的社区领导者。所有投诉都将被迅速而公正地审查和调查。

所有社区领导者都有义务尊重任何事件报告者的隐私和安全。

## 执行指南

社区领导者在确定任何违反本行为准则的行为的后果时，将遵循这些社区影响指南：

### 1. 纠正

**社区影响**：使用不适当的语言或其他在社区中被认为不专业或不受欢迎的行为。

**后果**：社区领导者发出私人的书面警告，阐明违规行为的性质以及为什么该行为不适当。可能会被要求公开道歉。

### 2. 警告

**社区影响**：通过单次事件或一系列行为违反规定。

**后果**：发出警告，说明继续此类行为的后果。在指定时间内禁止与涉及的人员互动，包括与执行行为准则的人员的不请自来的互动。这包括避免在社区空间以及外部渠道（如社交媒体）中的互动。违反这些条款可能导致临时或永久禁言。

### 3. 临时禁言

**社区影响**：严重违反社区标准，包括持续的不当行为。

**后果**：在指定时间内临时禁止与社区进行任何形式的互动或公开交流。在此期间，不允许与涉及的人员进行公开或私下的互动，包括与执行行为准则的人员的不请自来的互动。违反这些条款可能导致永久禁言。

### 4. 永久禁言

**社区影响**：表现出违反社区标准的行为模式，包括持续的不当行为、对个人的骚扰，或对某些类别的个人的攻击或贬低。

**后果**：永久禁止与社区进行任何形式的公开互动。
